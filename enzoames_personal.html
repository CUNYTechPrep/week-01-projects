Web Scraping – Scrapy Framework, Python

The Scrapy Framework allows you to construct spiders which scrape the desired information from any website. A spider is provided with a start_url, then follows specific xpath or css selector to extract data. Spiders can dynamically follow urls contained in start_url. Finally, all data extracted is saved to a Json file.
